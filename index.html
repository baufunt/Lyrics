Getting started
Up and running in minutes.
1

Invite Lyrics to your server

Click "Add to Discord" and authorise Lyrics on your server. Make sure to grant the recommended permissions so all features work correctly.

2

Configure your setup

Use ,enable welcome #channel, ,modlogs enable #channel and ,setprefix to tailor Lyrics to your server's needs.

3

Build your first embed

Head to the Embed Builder on this page, design your welcome message or announcement, copy the generated command and paste it in Discord.

4

You're done

Lyrics will handle the rest. Join the support server any time if you run into questions or want to suggest new features.

Questions
Frequently asked.
Is Lyrics free to use?
Yes, Lyrics is completely free. All commands and features are available to every server without any paid tier or subscription.
How do I change the bot prefix?
Use the ,setprefix [new prefix] command. The default prefix is a comma and each server can have its own unique prefix.
What permissions does Lyrics need?
Lyrics requests Administrator permission so all features work correctly. Make sure Lyrics' role is above other roles in your hierarchy for moderation features to work.
How do I use the Embed Builder?
Fill in the fields in the builder on this page, watch the Discord preview update live, then click "Copy Command" and paste the generated syntax into any channel where Lyrics is present.
Where do I get help or report a bug?
Join the support server at discord.gg/V3uXKHQvJu and open a ticket. The team responds quickly and all bug reports are tracked and fixed.
